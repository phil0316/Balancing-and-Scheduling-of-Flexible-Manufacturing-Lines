# 弹性产线配置决策工具

基于**双种群遗传算法（DPGA）** 的装配线平衡与人员配置优化工具。支持**Web 界面（Flask）**、**本地图形界面（Tkinter）** 和**命令行**三种使用方式，可自动生成包含甘特图、山积图、KPI 指标的 HTML 分析报告。

## 功能特点

- 根据年产量、工作日、班次、OEE 自动计算**目标节拍**
- 考虑装配线**紧前约束**，优化**工站数量**与**工序分配**
- 支持**工序相邻约束**（如某两道工序必须在相同或相邻工站）
- 优化目标：工站数最少 + 平滑指数（SI）最低 + 约束满足
- 输出 **线平衡率（LBR）**、**平滑指数（SI）**、**人力成本** 等关键 KPI
- 自动生成 **HTML 报告**，包含：
  - 关键指标卡片
  - 山积图（工站负荷 vs 节拍）
  - 工序甘特图（工序 → 工站分配）
  - 工序分配明细表

## 项目结构

├── app.py # Flask Web 入口

├── gui.py # Tkinter 本地图形界面

├── main.py # 命令行入口（支持交互/批处理）

├── optimizer.py # 遗传算法核心（DPGA + 解码 + 适应度）

├── report_gen.py # HTML 报告生成（SVG 甘特图 + Chart.js）

├── 遗传2.py # 单种群 GA 实验版本（仅供参考）

└── README.md # 项目说明

## 使用方式

### 1. Web 界面（推荐）

pip install flask numpy

python app.py

在浏览器打开 http://localhost:5000，填写生产参数，点击“开始优化”，等待 10~30 秒后自动在新窗口打开 HTML 报告。

### 2. 本地界面
 
pip install numpy

python gui.py

弹出窗口，运行即可

### 3. 命令运行

python main.py
